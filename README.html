Manual de uso

OBS: se debe copiar toda la carpeta ImgAtms a la máquina local donde se va a ejecutar y ejecutar el .exe desde la máquina.

  1. Primero se debe seleccionar la acción a realizar

  2. Luego se debe introducir el nombre de la imagen.

    Obs: si se va a copiar una imágen la imágen previamente debe estar en la carpeta Img en la raiz de este proyecto.

  3. Por último se seleccionan a que complejos copiar o de que complejos eliminar la imagen.

  4. Se presiona en el botón Ejecutar y se espera por los resultados.
